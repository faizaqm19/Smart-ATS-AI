# Smart ATS AI

AI-Powered Resume Screening, Skill Matching & Candidate Ranking System

---

## 📌 Overview

Smart ATS AI is an intelligent recruitment system designed to automate resume screening and candidate ranking using Artificial Intelligence and Natural Language Processing (NLP).

The system analyzes resumes against job descriptions, calculates ATS compatibility scores, identifies matched and missing skills, and ranks candidates based on suitability for the role.

This project helps recruiters save time, reduce hiring bias, and improve candidate selection accuracy.

## 🧠 How Smart ATS AI Works

### 1. Enter Job Description
The recruiter pastes a job description containing required skills, experience, and qualifications.

### 2. Upload Resume PDFs
Users upload one or multiple resumes in PDF format.

### 3. AI Matching Engine
The system extracts resume text using NLP and compares it with the job description.

### 4. ATS Score Generation
An ATS compatibility score is generated using semantic similarity and skill matching.

### 5. Candidate Ranking
Candidates are ranked automatically based on their match score.
